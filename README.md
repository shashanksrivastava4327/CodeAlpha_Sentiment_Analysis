📌 Project Overview

This project focuses on analyzing customer reviews from Amazon products using Natural Language Processing (NLP) techniques. The objective is to determine whether a review expresses a Positive 😊, Negative 😞, or Neutral 😐 sentiment.

By transforming unstructured text into meaningful insights, this project helps understand customer opinions, satisfaction levels, and overall product perception.

---

🎯 Objectives

✔ Analyze customer review text data

✔ Classify reviews into Positive, Negative, and Neutral sentiments

✔ Visualize sentiment distribution

✔ Generate meaningful insights from customer feedback

✔ Apply NLP techniques using Python

⚙️ Project Workflow

1️⃣ Data Collection

- Loaded Amazon review dataset using Pandas.

2️⃣ Data Cleaning

- Removed missing values.
- Processed review text.

3️⃣ Sentiment Analysis

- Applied TextBlob polarity scores.
- Classified reviews into:
  - Positive
  - Negative
  - Neutral

4️⃣ Data Visualization

- Pie Chart
- Bar Chart
- Word Cloud

5️⃣ Insight Generation

- Evaluated overall customer sentiment.
- Identified dominant customer opinions.

---

📊 Visualizations

🥧 Sentiment Distribution Pie Chart

📊 Sentiment Count Bar Chart

☁️ Word Cloud

Highlights the most frequently used words in customer reviews.

📈 Results

The analysis revealed that:

✅ Majority of customer reviews are Positive.

✅ Negative reviews form only a small portion of total feedback.

✅ Customers generally express satisfaction with products.

✅ Word Cloud highlights commonly discussed product features and experiences.
